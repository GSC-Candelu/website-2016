# gscandelu-static
Website for Gruppo Sportivo &amp; Culturale (Candelu', Italy)
Static version

### What is this?

This is a static site and serves as the website for an Italian non-profit organization
the Gruppo Sportivo &amp; Culturale of Candelu' (Treviso).

### Stack
- Bootstrap 3.3.6
- Fontawesome 4.5.0
- jQuery 1.12.2
